Detailed Description

GPIO interface for LED control and button handling in the Smart Home Heater project.

This module provides functions for initializing and controlling GPIO-connected LEDs and buttons. It supports system interaction via physical buttons and visual status indication through LEDs.

Authors
Duarte Caldeira, Tiago Saraiva
